 **Food Delivery App UI | Coding Samurai Internship**



**Project Overview**

The **Food Delivery App UI** is a modern, user-friendly mobile application interface created as part of the 

Coding Samurai UI/UX Internship (Task 2).  

This design focuses on delivering a smooth ordering experience by providing clear navigation, attractive visuals, and easy access to food items and order details.



The project demonstrates:

\- Clean UI layout  

\- Strong color hierarchy  

\- Organized wireframes  

\- Smooth user flow from login ‚Üí order ‚Üí delivery  



---



 **Objective of the Project**

The main goal of this UI design is to create a \*\*simple and intuitive food delivery experience\*\* that allows users to quickly browse dishes, add them to their cart, and place an order without confusion.



This UI is based on a typical food delivery flow:

1\. User opens the app  

2\. Logs in or creates an account  

3\. Views categories \& recommended dishes  

4\. Adds items to the cart  

5\. Confirms the order  

6\. Tracks delivery in real time
